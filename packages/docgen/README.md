## About

`@reciple/docgen` A documentation json generator and parser for reciple.

## Installation

```bash
npm install @reciple/docgen
yarn add @reciple/docgen
pnpm add @reciple/docgen
bun install @reciple/docgen
deno install npm:@reciple/docgen
```
