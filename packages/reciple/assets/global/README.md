# reciple

This is a [reciple](https://reciple.js.org) project.

## Overview

To see the documentation, visit [reciple.js.org](https://reciple.js.org).

### Development

First, start the bot in development mode:

```bash
npm run dev
```

To start the bot in production mode, build the modules first:

```bash
npm run build
```

Then, start the bot:

```bash
npm run start
```

> You can use your own package manager instead of `npm`. For example, you can use `yarn` or `pnpm`.
